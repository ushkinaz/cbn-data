Quickstart Command Line

Because typing by hand is how you get eaten by an angry JSON parser.

curl -s https://cbn-data.pages.dev/builds.json | jq '.[0]' curl -s https://cbn-data.pages.dev/data/2024-01-10/all.json.br --output all.json.br

Is this safe?

It is as safe as any bunker full of compressed game data. No zombies inside, just bytes.

Why so many files?

Because history matters, and also because future you will blame past you if you prune too hard.

Can I cache it?

Yes. Cache it like you cache canned beans. The headers are already doing most of the work.

Where did this data come from?

Direct from Cataclysm: Bright Nights releases, freshly washed and mostly sorted.
